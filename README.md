# Grocery Management System

A web-based Grocery Management System built using Node.js, Express, MongoDB, and EJS.  
This project allows users to browse grocery products, add them to a cart, place orders, and manage their profile.

---

## 🚀 Features

- User Registration and Login
- Product Listing
- Add to Cart
- Order Placement
- Order History
- User Profile Management
- Admin Product Management (Add/Edit/Delete)
- Secure authentication

---

## 🛠️ Tech Stack

- Backend: Node.js, Express.js
- Database: MongoDB with Mongoose
- Frontend: EJS, TailwindCSS
- Authentication: Session / Middleware
- Version Control: Git & GitHub

## ⚙️ Installation

1. Clone the repository
   git clone https://github.com/yourusername/grocery-management.git

3. Go into the project directory
   cd grocery-management

5. Install dependencies
   npm install

7. Start the server
   node server.js

or

npm start

---

## 🌐 Local Development

After running the server, open your browser and go to:
